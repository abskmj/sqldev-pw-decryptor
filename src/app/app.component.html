SQL Developer Password Decryptor
Passwords are stored encrypted in the connections.xml file which can be searched for in these locations:
  • Windows: C:\Users\<USER>\AppData\Roaming\SQL Developer\
  • Linux: ~/.sqldeveloper/ 

// connections.xml
<StringRefAddr addrType="password">
<Contents>F35q3vdbVrI=</Contents>
</StringRefAddr>
Encryption key by default is a machine-unique value db.system.id in the product-preferences.xml which for in above mentioned locations.

Oracle SQL Developer allows a user to associate passwords with connections so that the user doesn't have to enter the password each time he opens a connection. Of course, these passwords need to be stored somewhere. SQL developer stores them in an encrypted form, but it is possible to decrypt them.
